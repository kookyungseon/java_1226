3. 제어문자(이스케이프 문자)
따옴표 안에서 사용되며 미리 예약된 특수한 기능을 하는 문자 
\ (백슬래시, 역슬래시)를 사용한다.

		\n: 줄바꿈
		\t : 일정 간격을 띄워준다 (tab)
		\” : 문자열 안에 큰 따옴표를 표현
		\’ : 문자열 안에 작은 따옴표를 표현
		\\ : 문자열 안에 역슬래시 표현


4. 출력 메소드 

괄호 안에 있는 값을 콘솔창에 출력. 

- println()
	전달된 값을 출력하고 줄바꿈이 된다 
- print()
	전달된 값을 출력하고 줄바꿈이 되지 않는다
- printf()
	int a = 1;
	“1”
	형식(서식 format)에 맞는 값을 출력하고 줄바꿈이 되지 않는다 

형식지정자(서식문자)
- 반드시 따옴표안에서 작성한다.
- 출력 형식을 지정한다 
- %로 시작하며 printf()에서 사용한다
- ‘,’ 뒤에 나오는 값이 형식 지정자 위치로 들어가게 한다.

자료형 		형식지정자(서식문자)
int		%d			decimal(10진수 정수)
double	%f			float(실수)
char		%c			character(문자)
String		%s			String(문자열)


5. 형변환 (Type Casting)
- 자료형을 다른 자료형으로 변환하는 것
- 형변환은 숫자형 기본 자료형 간에만 가능 (정수, 실수)

1) 자동 형 변환 (Promotion)
		-작은 byte의 값을 큰 byte의 변수에 저장하면 자동적으로 형변환이 된다.
		- 정수 + 정수 = 정수 
		  정수 + 실수 = 실수 
		- 문자(char)는 정수(int)로 자동 형 변환된다.
		  자바에서 문자를 저장할때 유니코드의 정수값을 저장한다. 

2) 강제 형 변환(Casting)
		- 큰 byte의 값을 작은 byte의 변수에 저장하려면 강제로 형변환 해야한다.
		- 실수 -> 정수 
		- (자료형) 값;
		ex) (double) 10;
